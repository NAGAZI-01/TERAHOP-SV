[x] ตรวจสอบเนื้อหาไฟล์ index.html ปัจจุบัน
[x] หาช่องกรอกรหัสพนักงานในฟอร์ม
[x] เพิ่มฟังก์ชันการกรอกชื่อพนักงานในช่องเดียวกับรหัส
[x] เพิ่มฟังก์ชันตรวจสอบรหัสพนักงานซ้ำก่อนส่งข้อมูล

[x] อัพเดทรูปแบบรหัสพนักงานเป็น C25XXX
[x] ปรับปรุงการส่งออกไฟล์ Excel ให้สวยงาม
[x] จัดรูปแบบตารางใน Excel ให้สวยงาม
[x] เพิ่มสีสันและการจัดรูปแบบข้อความ
[x] ทดสอบการส่งออกไฟล์ Excel
[x] แก้ไขปัญหาเวลาไม่เดินและฟังก์ชันเสีย
[x] ตรวจสอบไฟล์ index.html ที่เกิดข้อผิดพลาด
[x] ซ่อมแซมฟังก์ชันที่ใช้งานไม่ได้
[x] ตรวจสอบปัญหา URL และพารามิเตอร์ที่ผิดปกติ
